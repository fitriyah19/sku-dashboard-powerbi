# Power BI SKU Governance Dashboard

## Overview
A Power BI dashboard designed to monitor material creation, governance compliance,
and business unit performance.

> ⚠️ Data shown is anonymized and for demonstration purposes only.

## Business Problem
Multiple business units create materials independently, leading to governance
and SKU inconsistencies.

## Key Metrics
- Number of materials created by business unit
- Monthly creation trends
- Governance compliance status
- Material type distribution

## Dashboard Pages
- Overview
- Business Unit Performance
- Material Details

## Tools Used
- Power BI
- Power Query (ETL)
- DAX
- Excel (source data)

## What I Did
- Data cleaning & transformation
- KPI design aligned with business needs
- Interactive filtering & drill-through
- Dashboard UX optimization
